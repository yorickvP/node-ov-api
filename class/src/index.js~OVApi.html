    public getTransactions(mediumId: Number, startDate: String, endDate: String, offset: Number): Promise source

    Get transactions between two dates.

    Params:

    NameTypeAttributeDescription
    mediumId Number

    Unique card id. You can pull this from getCards(). This field is required.
    startDate String
    • optional
    • default: current_date

    The date were the server should start looking. Correct syntax = YYYY-MM-DD.
    endDate String
    • optional
    • default: current_date

    The date were the server should stop looking. Correct syntax = YYYY-MM-DD.
    offset Number
    • optional
    • default: 0

    How many records the server should skip.

    Return:

    Promise

    Resolve returns transactions. Reject returns an error message that occured while fetching the transactions.
